# cordova-plugin-padmenu-demo
Demonstrates cordova-plugin-padmenu features

## padmenu plugin Demo

Showcases cordova-plugin-padmenu plugin features.

This app demonstrates the following:

* Adding menus and menu items
* Removing menus and menu items
* Adding keyboard shortcuts
* Menu item handling


The easiest way to run the demo is to download (or clone) the repo and run the project in Demo from Xcode.
